# 项目简介
## 核心功能
- **旅游景点推荐**：系统基于协同过滤算法，自动为用户推荐旅游景点。
- **用户个人中心**：用户登录后可查看个人信息、修改密码、预定景点、管理发布和收藏。
- **管理员功能**：管理员可以管理用户、景点类型、旅游景点、预定、攻略、论坛等。
- **系统管理**：包括系统首页、个人中心、举报记录等。

## 适用场景
- 旅游爱好者：发现并预订心仪的旅游景点。
- 旅游公司：为顾客提供个性化推荐服务。
- 管理者：有效管理旅游资源和用户数据。

## 技术特点
- **技术架构**：采用B/S架构，基于MVC模式，使用Java、SpringBoot、Vue等技术栈。
- **开发环境**：IDEA、JDK1.8、Mysql、Maven、Navcat。
- **数据管理**：使用SQL文件存储数据，支持Navcat等数据库管理工具操作。
